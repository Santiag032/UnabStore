🛍️ UnabStore

UnabStore es una aplicación de comercio electrónico desarrollada para ofrecer una experiencia de compra sencilla, rápida y moderna. Diseñada pensando en la comunidad universitaria, UnabStore permite explorar productos, agregarlos al carrito y gestionar pedidos de forma eficiente y segura.

🚀 Características principales

Catálogo de productos con imágenes, descripciones y precios.

Funcionalidad de carrito de compras.

Sistema de autenticación para usuarios.

Gestión de pedidos y compras.

Interfaz intuitiva y responsive para una experiencia óptima en cualquier dispositivo.
